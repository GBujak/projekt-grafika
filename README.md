# Projekt - grafika komputerowa

Projekt używa <https://mesonbuild.com> do kompilacji.

### Wymagania:

* biblioteka `sdl2`
* python3 + meson
* kompilator C++17

### Kompilacja

```sh
meson builddir
cd builddir
meson compile
```
